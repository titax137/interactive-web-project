# Interactive Hand Tracking Sci-Fi Visual

A real-time interactive visual project using webcam hand tracking, glowing beams, and futuristic UI.

## Demo

![Demo](demo.gif)

## Features
- Two-hand tracking
- Finger-connected glowing strands
- Moving scanner particles
- Sci-fi hologram visuals
- Built with p5.js + MediaPipe
